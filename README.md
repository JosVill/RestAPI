## Dependencias

- jjwt

```

	<groupId>io.jsonwebtoken</groupId>
	<artifactId>jjwt</artifactId>
	<version>0.9.1</version>
</dependency>
```

**User**

- lastname: String
- email: String
- phone: String
- password: String
